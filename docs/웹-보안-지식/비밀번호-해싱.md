### 🔍 개념

- 비밀번호를 안전하게 저장하기 위한 해싱 기법
- 단순 해시만 사용하면 무차별 대입 공격과 레인보우 테이블 공격에 취약
- 솔트(salt)와 느린 해시 알고리즘을 사용해 보안 강화

---


### 주요 알고리즘

- **bcrypt**
    - Blowfish 기반
    - 비용 인자(cost factor)로 연산 난이도 조절 가능
    - 자동 솔트 생성 및 저장 포함
- **scrypt**
    - 메모리 사용량을 조절해 GPU/ASIC 공격 방어 강화
    - 비용과 메모리 비용 조절 가능

---


### ⚠️ 주의사항

- 솔트 없이 해시하지 말 것
- 적절한 비용 인자 설정으로 느린 해시 구현
- 기존 해시 방식 변경 시 마이그레이션 전략 필요
