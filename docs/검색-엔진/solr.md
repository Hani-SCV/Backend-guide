### 🧠 개념

- Apache Lucene 기반의 **오픈소스 검색 플랫폼**
- 대용량 데이터에 대한 풀텍스트 검색, 색인, 분석 기능 제공
- RESTful API 지원, Java 기반

---


### ⚙️ 특징

- 분산 검색과 색인 지원 (SolrCloud)
- 다양한 쿼리 파서 및 필터 제공
- 풍부한 텍스트 분석기와 토크나이저 포함
- 복잡한 검색 요구사항 처리 가능
- Kibana 같은 시각화 도구와 연동 가능

---


### ✅ 장점

- 확장성 뛰어나고, 대용량 데이터 처리에 적합
- 설정과 커스터마이징 용이
- 다양한 언어와 포맷 지원
- 활발한 커뮤니티와 풍부한 문서

---


### ⚠️ 단점

- 초기 설정과 튜닝에 시간과 노력이 필요
- 운영 복잡도 있음
- Elasticsearch보다 생태계가 다소 좁음 (최근 기준)

---


### 🛠 활용 사례

- 웹사이트 검색엔진
- 전자상거래 상품 검색
- 기업 내 문서 검색 시스템
