### 🧠 개념

- 오픈소스 **분산 검색 및 분석 엔진**
- 대용량 실시간 데이터 검색, 로그 분석, 모니터링 등에 최적화
- Apache Lucene 기반, JSON 형태로 데이터 저장 및 검색

---


### ⚙️ 특징

- 분산형 아키텍처로 확장성 우수
- RESTful API 지원
- 풀텍스트 검색, 복합 쿼리, 집계 기능 제공
- 실시간 색인 및 검색 가능
- Kibana와 함께 시각화 도구로 자주 사용

---


### ✅ 장점

- 빠른 검색 속도 및 복잡한 쿼리 처리
- 스케일 아웃 가능 (노드 추가 용이)
- 다양한 데이터 타입과 쿼리 지원
- 로그, 메트릭, 텍스트 검색에 적합

---


### ⚠️ 단점

- 운영과 설정이 복잡할 수 있음
- 데이터 일관성 문제 발생 가능 (eventual consistency)
- 대규모 클러스터 관리 어려움
- 쓰기 작업이 상대적으로 느릴 수 있음

---


### 🛠 활용 사례

- 로그 및 이벤트 데이터 분석 (ELK 스택)
- 상품, 문서, 게시글 등 검색 서비스
- 추천 시스템, 데이터 탐색 도구
