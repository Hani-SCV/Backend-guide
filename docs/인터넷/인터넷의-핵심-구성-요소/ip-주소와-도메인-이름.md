### **IP 주소 (Internet Protocol Address)**

- **IP 주소**는 인터넷에 연결된 모든 기기들이 고유하게 가지는 주소입니다. 네트워크에서 **데이터를 정확하게 전달**하는 데 필요합니다.
    - 예: 웹 서버에 접근하려면 해당 서버의 IP 주소가 필요합니다.
- **형식**:
    - **IPv4**: `192.168.1.1` (숫자와 점으로 구성)
    - **IPv6**: `2001:0db8:85a3:0000:0000:8a2e:0370:7334` (16진수와 콜론 사용)

---


### **도메인 이름**

- IP 주소는 사람이 기억하기 어렵기 때문에, **도메인 이름**으로 변환하여 사용합니다.
    - 예: `www.google.com` → `142.250.190.78`
- **DNS (도메인 네임 시스템)**:
    - DNS는 도메인 이름을 **IP 주소로 변환**합니다. 사용자가 도메인 이름을 입력하면 DNS가 이를 찾아 해당 서버로 요청을 전달합니다.

---


### **추가 정보**

1. **도메인 이름의 구조**:
    - `www` → 서브도메인
    - `google` → 도메인 이름
    - `.com` → 최상위 도메인 (TLD)
2. **IP 주소와 도메인 이름의 관계**:
    - 하나의 도메인 이름은 **여러 IP 주소**에 연결될 수 있습니다. (예: 부하 분산)
    - 하나의 IP 주소가 **여러 도메인**을 가질 수 있습니다. (예: 가상 호스팅)
